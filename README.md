# Big Data Engineering: Spark, Dask & AWS

A self-paced interactive learning course for geospatial data engineers.  
**Live site →** https://firmanhadi21.github.io/spark-dask-aws-course

---

## What's inside

| Track | Lessons | Duration |
|---|---|---|
| ⚡ Apache Spark | 3 lessons | ~60 min |
| 🔀 Dask | 3 lessons | ~60 min |
| ☁️ AWS (S3, EMR, Lambda, Athena) | 3 lessons | ~65 min |

Each lesson includes:
- Plain-language explanation with geospatial analogies
- Working Python code examples
- Knowledge check quiz
- Progress tracking

Plus a **4-week study roadmap** and **quick reference cheatsheets**.

## Built with

- Pure HTML/CSS/JavaScript — no frameworks, no dependencies
- Hosted on GitHub Pages

## How to run locally

```bash
git clone https://github.com/firmanhadi21/spark-dask-aws-course
cd spark-dask-aws-course
# Open in browser
open index.html
```
